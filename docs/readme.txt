

Instrucciones para ejecutar el proyecto hotelandes del grupo A-12

Autores: Juan Diego Yepes - j.yepes
		 Sergio Pardo - s.pardog
		 
1. Descargue y descomprima el .zip del repositorio del proyecto asociado al commit de la entrega de la iteracion 2
2. Importe el proyecto a eclipse o a su IDE de preferencia
3. Edite el archivo persisence.xml y cambie las credenciales para acceder a la base de datos de Oracle
4. En la carpeta docs del proyecto Java dentro de la carpeta "documentos entrega" se encuentran las sentencias SQL para la creacion y poblacion de tablas
5. Copie las sentencias SQL para la creacion de tablas y ejecutelas
6. Copie las sentencias de poblacion de tablas únicamente hasta consumo servicio, no se hizo uso de las sentencias para poblar planes de consumo
7. Corra la aplicación desde la interfaz y pruebe las distintas opciones de agregación. 
8. Puede revisar que se agregen correctamente los datos a través de su conexión con la base de datos de Oracle.
